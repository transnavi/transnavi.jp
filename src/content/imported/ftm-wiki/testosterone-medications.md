テストステロン（男性ホルモンのひとつ）のお薬には、注射やジェル、飲み薬など、いくつかのかたちがあります。この記事では、それぞれのお薬が「どうしてそういう形になっているのか」という、からだとお薬の仕組みをやさしく紹介します。どのお薬を、どれくらい使うかは、人それぞれちがいます。かならず医師と相談しながら決めていきます。

## エステル（酯化）って何だろう

注射で使うテストステロンの多くは、「エステル化（酯化反応）」という工夫がされています。エステルとは、カルボン酸（酸の一種）とアルコールが反応してできる物質のことです。テストステロンの分子にエステルの部分（酯基）をくっつけると、油にとけやすくなります。油にとけやすくなると、注射した場所から血液のなかへ、テストステロンがゆっくり出ていくようになります。

テストステロンにつけられるエステルには、いろいろな種類があります。

| 丙酸 | Propanoic acid（プロピオン酸） |
| 己酸 | Hexanoic acid（ヘキサン酸） |
| 庚酸 | Heptanoic acid（ヘプタン酸／エナント酸） |
| 十一酸 | Undecanoic acid（ウンデカン酸） |
| 癸酸 | Decanoic acid（デカン酸） |
| 环戊丙酸 | （シクロペンチルプロピオン酸） |
| 异己酸 | （イソヘキサン酸） |
| 苯丙酸 | （フェニルプロピオン酸） |

エステルどうしのいちばんのちがいは、エステルの部分をつくっている「炭素のくさり（炭素鎖）」の長さです。

たとえば、十一酸テストステロン（ウンデカン酸テストステロン、C30H48O3）は、テストステロン（C19H28O2）に、炭素が11個の長さのエステル（酯基）をくっつけたものです。エステルをつけると「分配係数」（油へのとけやすさと水へのとけやすさの比）が大きくなり、これがお薬がからだのなかにどれくらいの長さとどまるかを決めます。

ほかにも、次のようなエステルがあります。

- 環戊丙酸テストステロン（TC、C27H40O3）
- 庚酸テストステロン（TE、C26H40O3）

## なぜエステルをつけるの

もしエステルをつけずに、水にうかせただけのテストステロンを注射すると、分配係数が小さいため、テストステロンが一気に血液のなかへ入ってしまいます。そうすると、血中の濃度が急に高くなりすぎる「ピーク」ができ、そのあとすぐにからだから消えてしまいます。これだと、毎日（または1日に何回も）、少しずつ入れつづけなければならなくなります。

エステルをつけると、血液のなかの濃度を安定させやすくなります。エステルの部分が大きく長いほど、油にとけやすくなり、分配係数が高くなって、お薬がゆっくり長く出るようになります。そのぶん、ピーク（濃度のいちばん高いところ）も低くおさえられます。

注射した場所から血液のなかへ入ったテストステロンのエステルは、「加水分解」という仕組みで、エステラーゼという酵素によって、テストステロンと脂肪酸とに分けられます。こうしてはじめて、からだが使えるかたち（生物学的に利用できるかたち）にもどり、血液検査でも見えるようになります。

## 飲み薬（口服）のテストステロン

テストステロンを飲み薬にするには、分子をすこし作りかえる工夫が必要です。理由は「初回通過効果（首過効果）」とよばれる仕組みにあります。口から飲んだお薬は、胃や腸から吸収されたあと、門脈という血管を通っていったん肝臓へ運ばれます。そして全身にいきわたる前に、肝臓で大きく分解されてしまいます。テストステロンをそのまま飲んでも、こうしてほとんど効かなくなってしまうのです。

これまでに、肝臓で分解されにくいように作りかえたお薬（メチルテストステロンなどのAAS系のもの）も使われてきました。ただし、これらは健康へのリスクがあるため、いまではあまりすすめられていません。

いまの飲み薬でいちばん新しいのが、十一酸テストステロン（ウンデカン酸テストステロン）のソフトカプセルです。このエステルを油酸（オレイン酸）にとかしてあり、小腸からリンパ系に吸収されるように作られています。リンパ系を通ることで、肝臓への負担を減らすことができ、メチルテストステロンよりも安全な飲み薬とされています。

## おぼえておきたいこと

- テストステロンのお薬には、注射・ジェル・飲み薬など、いろいろなかたちがあります。それぞれ、からだへの入り方や、効きめの続き方がちがいます。
- 注射薬の多くは「エステル化」という工夫で、ゆっくり長く効くようになっています。
- どのお薬が合うか、どれくらいの量にするかは、人によってちがいます。からだの状態をみながら、医師と相談して決めていきます。
- 自分だけの判断でお薬を使ったり、量を変えたりしないでください。気になることがあれば、まず医師に相談しましょう。
