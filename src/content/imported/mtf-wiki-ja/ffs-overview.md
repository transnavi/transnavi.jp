> **大切なこと**　日本では、ホルモン療法や手術は医師のもとで受けられます。この記事は、からだの仕組みや選択肢を知るための情報で、自己判断での使用・調整をすすめるものではありません。実際の治療・お薬・手術は、かならず医師に相談してください。

顔面女性化手術（FFS, Facial Feminization Surgery）は、顔の男性的な特徴をやわらげ、女性的な見た目を引き出すことを目的とした、形成・美容外科手術のひとつです。この記事は、FFSがどんなものなのかを知り、自分にとって必要かどうかを落ち着いて考えるための、入門的なまとめです。

## FFSとは

思春期以降の、出生時に男性と割り当てられたからだでは、顔にいくつかの男性的な特徴があらわれやすくなります。たとえば次のようなものです。

- 額（ひたい）が平たい、眉のあたりの骨（眉弓・びきゅう）が前に出ている
- 鼻すじが高い、鼻に出っぱり（わし鼻）がある、鼻の幅が広い、鼻先が下がっている、小鼻（鼻翼・びよく）が広い
- まぶたの構造（一重・三角の形・厚み）
- 頬骨（ほおぼね・頬骨弓）の幅が広い
- 下唇が広い、人中（鼻の下の溝）が長い、口角が下がっている
- あご（オトガイ・頤）の幅が広い、前に出ている
- ひげ　など

FFSは、こうした特徴のうち本人が気になる部分について、医師が一人ひとりの希望にあわせて手術内容を設計し、男性的な特徴を減らして女性的な表現を引き出していくものです。自分が自認する性別（女性）としての見た目に近づくことで、自己受容や性別移行を後押しする目的で選ばれることがあります。

## おもなFFSの手術項目

FFSは一つの大きな手術ではなく、いくつもの部位ごとの手術を組み合わせたものです。代表的な項目と、向いているとされる状態は次のとおりです。どの項目が自分に合うか、そもそも必要かどうかは、かならず医師と相談して決めます。

| 項目 | 向いているとされる状態 |
|------|------|
| 自家脂肪注入（額への脂肪注入を含む） | 額が平たい、眉弓が高い、頬（いわゆるアップルゾーン）がへこんでいる、顔が骨ばっている |
| 眉弓（びきゅう）降下術（前に出た額の骨板を切って、平らにし置き直す） | 額が平たい、眉弓が前に出ている |
| 重瞼術（じゅうけんじゅつ・二重まぶたの手術） | 一重、上まぶたのたるみ（いわゆる「三角眼」）、上まぶたの厚み（いわゆる「ぱっちりしないむくみ」） |
| 鼻の総合整形 | わし鼻、幅の広い鼻、鼻先の下がり、小鼻の広さ など |
| 唇まわりの総合整形（口角を上げる処置を含む） | 赤い部分が薄い、人中が長い、口角が下がっている |
| 頬骨弓（きょうこつきゅう）の縮小 | 頬骨弓の幅が広い |
| 下顎角（かがくかく・エラ）の骨切り | エラの幅が広い |
| オトガイ（あご）の縮小・短縮 | あごの幅が広い、あごが前に反り出ている |
| レーザー脱毛 | ひげ、ならびに体幹・手足などの毛 |
| 喉頭（のどぼとけ）の整形術 | のどぼとけが目立つ |

## FFSを考えるときに

「FFSをしてくれる医師をどう探すか」「自分にFFSは必要なのか」「どう選べばよいか」「FFSは自分に何をもたらすのか」。考えはじめると、こうした問いがいくつも出てきます。

FFSの本質は、美容・形成外科手術の一部です。ですから、一般的な美容医療を考えるときと同じ心構えが役に立ちます。

- **「女性化」は「きれいになること」とは同じではありません。** FFSをすれば必ずより良い見た目になる、というものではありません。
- まず、髪型や服装を変えることで印象が良くなるかどうか、試してみる価値があります。
- もし手術を考えるなら、自分がどこを、どんなふうに変えたいのかをはっきりさせておくことが大切です。仕上がりのイメージを持っておくとよく、絵を描いたりメイクで試したりして、どんな見た目になりたいかを確かめてみましょう。
- 自分の顔をじっくり見て考えると不安になることもありますが、一般的にどの部位を扱うのかを、自分なりに、あるいは誰かと一緒に、少し調べてから臨むほうが安心です。あわてて決めないようにしましょう。
- 大きな処置になりそうな術式を選ぶときは、自分を支えてくれる人や医師と、いろいろな選択肢をよく相談してください。
- 頬骨や眉弓など、ホルモン療法による脂肪のつき方の変化を待つことで印象が変わりうる部位については、まずその効果を待てるかどうかを相談してみる価値があります。
- 眉弓降下術（前に出た骨板を切って平らにし置き直す手術）の代わりに、額への脂肪注入で対応できないかを相談する、という選択肢もあります。

信頼できる医師を選ぶときは、その医師の手術後の仕上がりの例を見せてもらうとよいでしょう。

## 美容医療について

一般に「医療美容（美容医療）」を行っている医療機関では、レーザー脱毛、光治療（フォトフェイシャル）、傷あとの治療、ほくろの除去などができると考えられています。日本でも、これらは医師のもとで受けられます。
