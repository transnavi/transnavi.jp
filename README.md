# とらんすナビ

日本のトランスジェンダー情報ウィキ。公開ドメインは <https://transnavi.jp> です。

## 目的

とらんすナビは、日本で暮らすトランスジェンダー、ノンバイナリー、Xジェンダー、ジェンダーを探索中の人と、その支援者のためのオープンアクセス情報サイトです。

医療、法律、生活、地域情報を日本語中心に整理し、英語・中国語などの豊富な外部資料にも接続します。

## 技術スタック

- Astro
- Cloudflare Workers
- Cloudflare Wrangler

将来的に、GitHubを使わない提案フォーム、レビュー画面、リソース投稿機能を追加します。

## 開発

```sh
npm install
npm run dev
```

## ビルド
